# Angular-NgRx-Firebase
## Gorilla Logic SBC Building Rooms Visualizer

This is a demo project for learning about Firebase as and option for BaaS for quick access and broadcasting of information in realtime.
Technologies used: 

* Firebase: documentation and best practices as described in this article [Link to article](https://firebase.google.com/docs/firestore/best-practices)

* NgRx Store: using best practices as described in this article: [Link to article](https://itnext.io/ngrx-best-practices-for-enterprise-angular-applications-6f00bcdf36d7)

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 8.0.0.
